Dimensions API Lab: reusable code and solutions

This site contains read-only versions of the Jupyter notebooks included in the Dimensions API Lab Github repository.

The aim of these notebooks is to show how to carry out common scholarly data analytics tasks using the Dimensions Analytics API.

For more information about the Dimensions API language, please take a look at the official documentation. too.
