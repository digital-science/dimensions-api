Dimensions API Lab: reusable code and solutions

This site contains a collection of Jupyter notebooks showing how to carry out sophisticated scholarly data analytics tasks using the Dimensions APIs.

Note: the HTML versions of the Jupyter notebooks have been generated automatically from the Dimensions API Lab Github repository, where the source code for these examples is available for further inspection and download.

If you want to find out more about the Dimensions Analytics API language, please take a look at the official documentation.
